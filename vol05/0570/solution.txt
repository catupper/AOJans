問題番号:AOJ0570
問題名:ジグザグ数

解説:
dp[keta][last][mod][zigzag][less]を
{
上からketa桁目まで決めている
keta桁目はlastである
上keta桁をMで割ったあまりがmod
keta - 1桁目からketa桁目に値が増加している(zigzag = 0)か減少している(zigzag = 1)か
上keta桁はA(もしくはB)と同じ(less = 0)か、より大きい(less = 2)か、より小さい(less = 1)か。
}
という状態を表すメモテーブルとしてメモ化再帰する。
これによって1以上A未満の条件に合うジグザグ数と1以上B以下の条件に合うジグザグ数がわかるので差を取れば良い。

アルゴリズム:
DP

オーダー:
O(KM)
K = 桁数
M = 剰余の法
